Descrição de tegs

  • TABLE = Tag para criar tabelas
  • TH = Cria células principais
  • TR = É o pai do (th e tr)
  • TD = Criar as demais células
  • THEAD = Serve para organização (Cabeçalho)
  • TBODY = Serve para organização (Corpo da tabela)
  • TFOOT = Serve para organização (Parte final da tabela)
  • ALIGN = Serve para alinhar
  • BORDER = Criar uma linha de borda
  • CAPTION = Criar um título principal da tabela
  • ROWSPAN = Serve para mesclar células da coluna
  • COLSPAN = Serve para mesclar células da linha
  • WIDTH = Largura de um componente html
